Built in Pure Rust — Zero Dependencies
SQLite philosophy applied to AI infrastructure · 16 crates · ~22-week build order · sovereign, auditable, portable
The SQLite principle: no external dependencies. SQLite is the most deployed database on Earth — not because it's the fastest, but because it runs anywhere, answers to no one, and has never been broken by a supply-chain compromise.

ATLAS makes the same choice: zero Rust crate dependencies. Zero means zero — no tokio, no tch, no candle, no ndarray. Every algorithm is written from scratch or ported from its original C/Python source into our codebase.

CUDA is a system runtime, not a crate. It is called via raw extern "C" FFI from build.rs + kernels/*.cu. No cudarc wrapper, no cuBLAS Rust binding. The same principle that makes SQLite trustworthy, applied to the GPU.
16 Crates — Workspace Structure
atlas-core
Error types, traits, config primitives. The foundation everything else imports.
Write from scratch — trivial
atlas-tensor
Tensor{data,shape}, matmul, element-wise ops, CUDA FFI bridge. The seed of everything.
Write from scratch + CUDA kernels
atlas-grad
Autograd tape. Forward + backward pass through any computation graph. The hard part.
Port micrograd concept (~2K LOC)
atlas-optim
AdamW, cosine LR scheduler, weight decay. Standard algorithms — ~300 LOC.
Write from standard algorithm
atlas-quant
INT4 / INT8 quantization. QLoRA dequant for 24GB VRAM training of 7B model.
Port bitsandbytes quant logic
atlas-model
Transformer: MultiHeadAttn, FFN, RMSNorm, RoPE embeddings. OLMo 3 architecture config + weights loader.
Port OLMo 3 architecture spec
atlas-tokenize
BPE tokenizer. Encode/decode. Vocabulary management. Load OLMo tokenizer files.
Port sentencepiece BPE algorithm
atlas-palace
GraphPalace engine — 36 methods, stigmergic navigation, pheromone deposit/decay, semantic search.
Strip PyO3 bindings — already Rust
atlas-trm
TRM-CausalValidator: 7M params, 2-layer MLP, z=net(x,y,z)×6 recursions. Quality Gate 6.
Implement from arXiv:2510.04871 (~500 LOC)
atlas-causal
PC / FCI causal inference algorithms. The core of ASTRA's scientific rigor. ~800 LOC each.
Port from py-causal / tetrad Java source
atlas-bayes
Bayesian confidence scoring. P(H|E) updates. The quality gate backbone.
Port from fetch-agi Python source
atlas-astra
Full ASTRA OODA engine — Observe, Orient, Decide, Act. 89 endpoints. The largest port (~8K LOC).

atlas-corpus
LiveDiscoveryCorpus: harvester, 5 quality gates, pheromone-weighted batch sampler, Type 5 generator.

atlas-zk

atlas-http / atlas-json
HTTP client (NASA/WHO/WB APIs) via raw libc syscalls. JSON parser ported from source.
